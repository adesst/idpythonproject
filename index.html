    Tentang idpython

    Proyek ini dimulai karena belum adanya referensi terpusat untuk belajar bahasa pemrograman Python dari awal . Untuk garis besar proyek ini, kita akan mencontoh situs Django yang memiliki struktur yang sangat bagus sebagai portal informasi suatu project.

    • Penulisan tutorial-tutorial akan dilakukan dalam bentuk dokumen reStructuredText (plain text) sehingga semua bisa berkolaborasi untuk melengkapi maupun mengkoreksi panduan tersebut.
    • Blog post dalam website hanya akan berisikan informasi rilis Python maupun berita seputar Python Indonesia
    • Bagian komunitas akan berisikan post-post tentang python dalam bahasa Indonesia yang akan ditarik dari beberapa RSS website yang memiliki tutorial-tutorial maupun informasi Python dalam bahasa Indonesia.
